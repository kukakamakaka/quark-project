# Quark — AI Intelligence Layer

**Quark** — это высокотехнологичный лендинг, разработанный в рамках тестового задания на позицию Frontend Developer. Проект представляет собой платформу для децентрализованного ИИ и блокчейн-экосистемы с упором на производительность и современный визуальный стиль.

**[Live Demo](https://quark-project-nb5f.vercel.app/)**

---

### 🛠 Tech Stack (Технологический стек)

* **Framework:** Next.js 15+ (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React & React Icons
* **Fonts:** Montserrat (Headings) & Inter (Body)
* **Deployment:** Vercel

---

### ✨ Key Features (Основные возможности)

* **Responsive Design:** Полная адаптивность под Desktop, Tablet и Mobile (от 320px).
* **Performance:** Оптимизация изображений (WebP) и высокая скорость загрузки.
* **SEO Ready:** Настроены Meta-теги, OpenGraph и семантическая разметка (Header, Main, Footer).
* **Interactive UI:** Плавные анимации при скролле и интерактивные компоненты (Testimonials, Mobile Menu).
* **Clean Code:** Компонентный подход и отсутствие дублирования кода.

---

### 🚀 Getting Started (Запуск проекта)

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/kukakamakaka/quark-project.git](https://github.com/kukakamakaka/quark-project.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run development server:**
    ```bash
    npm run dev
    ```

---

### 📂 Project Structure (Структура проекта)

* `/app` — Роутинг и основные страницы.
* `/components` — Переиспользуемые UI компоненты (Hero, Advantages, Footer и т.д.).
* `/public` — Оптимизированные ассеты и изображения.
